# RLDatix-DataScience-Assessment
This is the work completed for a take assessment.

To run all of the code in the associated notebook, first complete the following. You should have Conda, Python, Jupyter, and pip installed on your machine first. 

1. In your CLI, from the directory of this project, run <conda create -n rltest python=3.10.18>
2. In your CLI, from the directory of this project, run <conda activate rltest>
3. In your CLI, from the directory of this project, run <pip install -r requirements.txt>
4. In your CLI, from the directory of this project, run <python3 -m jupyterlab> (depending on your installation of python and jupyter this may be slightly different)
